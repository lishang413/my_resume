# 前端在线简历

#### 介绍
一个适用于开发者的在线简历模板

#### 相关技术
Vue3+TypeScript

#### 使用说明
1. 执行以下命令克隆代码
```
git clone -b master https://github.com/lishang413/my_resume.git
```

2.进入项目目录，执行以下代码下载依赖
```
npm install
```

3.运行以下命令启动项目
```
npm run serve 或 vue serve
```

4.修改src/info/info.ts中内容，编辑为自己简历内容并预览

5.确认简历修改完成后，执行完成以下命令后，进入dist目录下即可发布属于自己的在线简历
```
npm run build 或 vue build
```
